 ⩥ Output Terminologies
  1. Positive Score: It is the number of positive words in your text.
  2. Negative Score: It is the number of negative words in your text.
  3. Subjectivity Score: This is the score that determines if a given text is objective or subjective.
  4. Polarity Score: This is the score that determines if a given text is positive or negative in nature. Range is from -1 to +1.
  5. Fog Index: This is the value which determines the readability of your text. The lesser the fog index, the better is readability.
  6. Complex Words: Words which contains more than two syllables are called complex words.
  7. Unidentified words: Words which are not registered or recorded in our data.

⩥ About :
Sentiment Analyzer is a web Application developed in Python using Flask Framework and SentimentAnalyzer API which is helpful to analyze the sentiment of a given text with accuracy of 93.46%. Also, none of your analyzed text are stored on the server which makes it more secure and anonymous.
